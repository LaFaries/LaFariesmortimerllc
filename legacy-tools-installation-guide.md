# Legacy & Family Tools - Installation Guide

## What is this?

A standalone desktop application for managing family legacy, memories, and estate planning. Runs entirely on your computer - no internet required, no data uploaded.

---

## Quick Start (Easiest Method)

### For Windows:
1. Download `legacy-family-tools-app.html`
2. Right-click the file → Open with → Choose your browser (Chrome, Edge, Firefox)
3. Bookmark the page for easy access
4. Done! The app is now running locally

### For Mac:
1. Download `legacy-family-tools-app.html`
2. Double-click the file (opens in your default browser)
3. Add to bookmarks for quick access
4. Done!

---

## Desktop Shortcut (Recommended)

### Windows:
1. Right-click `legacy-family-tools-app.html`
2. Create shortcut
3. Move shortcut to Desktop
4. Rename to "Legacy & Family Tools"
5. Double-click to launch anytime

### Mac:
1. Right-click `legacy-family-tools-app.html`
2. Make Alias
3. Move to Desktop or Applications folder
4. Rename to "Legacy & Family Tools"
5. Double-click to launch

## Features

### Family Tree Builder
- Create digital or printable family trees
- Track multiple generations
- Export as text file

### Legacy Letter Generator
- Write heartfelt letters to family
- Include values, life lessons, hopes
- Download as text file

### Memory Vault
- Store personal memories privately
- Add dates and categories
- Export all memories at once
- Delete individual items

### Estate Planning Checklist
- 10-step preparation guide
- Track completion progress
- Print checklist
- Save progress in browser

## Privacy & Data Storage

**ALL data stays on your computer**
- Nothing is uploaded to the internet
- No account required
- No tracking
- Data stored in browser's local storage

**To keep your data safe:**
- Don't clear your browser data/history
- Export/download your work regularly
- Keep downloaded files in a safe folder

---

## Backup Your Work

**Regular exports recommended:**
- Legacy letters → Click "Download Letter"
- Family trees → Click "Download as Text"
- Memories → Click "Export All as Text"
- Estate checklist → Click "Print Checklist" → Save as PDF

Store these files on:
- External hard drive
- Cloud storage (Google Drive, Dropbox)
- USB flash drive

---

## Troubleshooting

**App won't open:**
- Try a different browser
- Make sure file hasn't been corrupted
- Re-download if needed

**Lost my data:**
- Data only saved in browser's storage
- If you cleared browser data, it's gone
- Always export regularly!

**Can't download files:**
- Check browser's download settings
- Allow downloads from local files
- Try a different browser
